## Physical Security Tools
### Lock Picking
- **Lock picks**: Various types
- **Tension wrenches**: BOK and TOK
- **Bump keys**: For pin tumbler locks
- **Shims**: For padlocks
- **Decoders**: For combination locks

### Surveillance
- **Cameras**: IP and analog
- **DVR/NVR**: Recording systems
- **Hidden cameras**: Covert monitoring
- **Audio recording**: Voice capture
- **GPS trackers**: Asset tracking

### Access Control
- **Key cards**: RFID and magnetic
- **Biometrics**: Fingerprint, iris, face
- **PIN pads**: Code entry
- **Mantraps**: Dual-door entry
- **Turnstiles**: Controlled passage

### Testing Tools
- **Proxmark**: RFID cloning
- **GPIO tools**: Raspberry Pi attacks
- **USB Rubber Ducky**: Keystroke injection
- **WiFi Pineapple**: Network attacks
- **HackRF**: SDR for signal analysis
