# Social Engineering Reference

## Social Engineering Vectors
### Phishing
- **Email Phishing**: Mass email campaigns
- **Spear Phishing**: Targeted attacks
- **Whaling**: Executive targeting
- **Clone Phishing**: Legitimate email replication
- **Vishing**: Voice-based phishing
- **Smishing**: SMS-based phishing

### Pretexting
- Create believable scenarios
- Build trust with targets
- Use authority and urgency
- Research targets thoroughly
- Maintain consistent cover story

### Baiting
- Leave infected media devices
- Use enticing file names
- Exploit curiosity
- Create physical traps
- Use USB drops

### Quid Pro Quo
- Offer services in exchange for info
- Impersonate IT support
- Use free offers as bait
- Create sense of obligation
- Exploit helpful nature

### Tailgating/Piggybacking
- Follow authorized personnel
- Hold doors open
- Use delivery伪装
- Impersonate employees
- Exploit politeness

## Social Engineering Techniques
### Psychological Principles
- **Authority**: Impersonate authority figures
- **Urgency**: Create time pressure
- **Scarcity**: Limited availability
- **Social Proof**: Others are doing it
- **Reciprocity**: Give to get
- **Commitment**: Small yeses lead to big yeses
- **Liking**: Build rapport
- **Fear**: Threaten negative outcomes

### Elicitation Techniques
- Open-ended questions
- Active listening
- Mirroring body language
- Building rapport
- Using flattery
- Creating discomfort
- Using silence

### Manipulation Tactics
- **Anchoring**: First number influences
- **Door-in-the-face**: Big request then small
- **Foot-in-the-door**: Small request then big
- **Low-ball**: Underquote then raise
- **Bait-and-switch**: Switch after commitment

## Social Engineering Prevention
### Awareness Training
- Regular security awareness training
- Simulated phishing campaigns
- Report suspicious activity
- Understand attack vectors
- Recognize red flags

### Technical Controls
- Email filtering and anti-phishing
- Multi-factor authentication
- Security awareness platforms
- Incident response procedures
- Regular security audits

### Organizational Controls
- Security policies and procedures
- Background checks
- Separation of duties
- Regular audits
- Incident response plan

## Social Engineering in Penetration Testing
### Planning
- Define scope and rules of engagement
- Identify target personnel
- Research organizational structure
- Develop pretexts
- Prepare materials

### Execution
- Conduct phishing campaigns
- Perform pretexting scenarios
- Test physical security

### Reporting
- Document successful attacks

- Conduct debrief sessions
- Create awareness materials
- Follow up with training
